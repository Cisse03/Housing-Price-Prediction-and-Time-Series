# Housing Price Prediction and Time Series Forecasting

Applied regression and time series analysis to investigate the factors that influence house prices and forecast future housing price trends.

## Project Overview

This project investigates the relationship between housing characteristics and sale prices using multiple linear regression. Several predictor variables were analyzed to determine which factors have the greatest impact on house prices. Model diagnostics and variable selection techniques were used to evaluate the quality of the regression model. In addition, time series analysis was performed to model and forecast future housing price trends using an ARIMA model.

## Research Question

Which housing characteristics have the greatest influence on house prices, and how well can regression and time series methods explain and forecast housing prices?

## Methods

The following statistical methods were used:

- Multiple Linear Regression
- Progressive Variable Selection
- Residual Diagnostics
- Variance Inflation Factor (VIF)
- Durbin-Watson Test
- Time Series Analysis
- Logarithmic Transformation
- ARIMA Forecasting

## Dataset

- Dataset: KC House Sales Dataset
- Response Variable:
  - House Price
- Predictor Variables:
  - Bedrooms
  - Bathrooms
  - Living Area
  - Lot Size
  - Floors

## Key Results

- Living area was the strongest predictor of house prices.
- Bathrooms also had a positive effect on predicting house prices.
- The regression model explained approximately **50.86%** of the variation in house prices.
- Variable selection identified living area, bedrooms, bathrooms, and lot size as the best subset of predictors.
- Model diagnostics showed that the regression assumptions were generally satisfied.
- The ARIMA(5,1,0) model indicated that house prices are expected to remain relatively stable over the short-term forecast period.
